
INSTRUKTIONER:
Spelet går ut på att förflytta karaktären till dörren. För att klara av leveln måste spelaren dock hämta nyckeln först. De gröna rektanglarna är fiender och kolliderar karaktären med dem så förlorar man ett liv och man får börja från startpositionen igen (man förlorar dock inte sina upplockade saker). Förlorar man alla sina (5) liv får man Game Over.

Karaktären styrs med piltangenterna, och om man plockar upp en kula (de röda cirklarna) så kan man skjuta på 'A'.

Spelet kan pausas med 'P' och för att se status-info för karaktären trycker man 'S'.
