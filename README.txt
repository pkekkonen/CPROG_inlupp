
INSTRUKTIONER:
Spelet går ut på att förflytta mus-karaktären till dörren. För att klara av leveln måste spelaren hämta nyckeln först. De gröna rektanglarna är fiender och går man på dem så förlorar man ett liv och får börja från startpositionen igen (man förlorar dock inte sina upplockade saker). Förlorar man alla sina (5) liv får man Game Over.

Spelaren styrs med piltangenterna, och om man plockar upp en kula (de blåa cirklarna) så kan man skjuta på 'A'.

Spelet kan pausas med 'P' och för att se status info för spelaren trycker man 'S'.
