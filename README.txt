
INSTRUKTIONER:
Spelet går ut på att förflytta karaktären till dörren. För att klara av leveln måste spelaren hämta nyckeln först. De gröna rektanglarna är fiender och kolliderar karaktären med dem så förlorar man ett liv och får börja från startpositionen igen (man förlorar dock inte sina upplockade saker). Förlorar man alla sina (5) liv får man Game Over.

Spelaren styrs med piltangenterna, och om man plockar upp en kula (de röda cirklarna) så kan man skjuta på 'A'.

Spelet kan pausas med 'P' och för att se status-info för spelaren trycker man 'S'.


Angående filer:

- samtliga bildfilers sökvägar är skrivna som "IMAGE_PATH/GameImages/bildens-filnamn" så det bara är att ersätta IMAGE_PATH med rätt sökväg.
- I System.cpp finns två sökvägar till fonter som eventuellt kan behöva justeras
